PROJECTS

Twitch TV Viewer

As an avid user of twitch.tv, I thought it would be helpful if I had a personalized list of my favorite streamers at the click of a button. I'm able to see if they're online, and by clicking their name, I can see detailed statistics about their stream such as what game they're playing, how long since the stream started, and a preview of what I can expect to see if I decide to view their stream directly on Twitch.tv. Additionally, this is designed responsively with custom breakpoints and no bootstrap at all.


Technologies used:
HTML5, CSS3, JS, JQuery, Twitch API, Sass, Responsive Design

OKLunch

This is my final project for my immersive coding experience at Dev Bootcamp - created in February 2016. It is a full stack application created in Ruby on Rails. The idea for OKLunch is essentially a Tinder-like app, except for lunch and two layers of matching - matching restaurants and matching with users. Users can register an account manually, or use a one-click registration via Facebook login.


Technologies used:
HTML5, CSS3, JS, JQuery, JQuery UI, OmniAuth API, Yelp API, Facebook API, Ruby on Rails, PostgreSQL, ActiveRecord, Heroku
